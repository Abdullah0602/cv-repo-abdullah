# CV Website

## Project Info
This is my personal CV website created as a Web development assignment. The CV includes a header with my contact information (phone, Gmail and GitHub), a main section with a short paragraph about me alongside my picture and sections for education, skills, contact and experience. Additional features include a **Contact Me form, Light/Dark mode and a Download CV** option.
